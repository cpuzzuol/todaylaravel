Things needed to install on Apache for production (worked on Ubuntu 20.04 with Laravel 8)

Laravel docs: https://laravel.com/docs/9.x/deployment
sudo apt-get install php7.4-curl
sudo apt-get install libapache2-mod-php7.4
sudo apt-get install php7.4-dom
